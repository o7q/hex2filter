# Overview
hex2filter takes in 3 values:
- Input Hex
- Max Loss
- Max Attempts

# Usage

## <b>Interface</b>
- **Input Hex** Input HEX value to be converted to a CSS filter
- **Max Loss** Max color loss value (default: 0.1)
- **Max Attempts** Max attempts to get a lossy result less than `Max Loss`
